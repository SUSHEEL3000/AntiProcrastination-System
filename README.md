# Anti-Procrastination System

A modern Android productivity application built using **Kotlin + XML Views** designed to help users overcome procrastination through streak tracking, habit reinforcement, focus sessions, and reward-based motivation.


## Features

### Don't Break The Chain

* Maximum 3 tasks per day
* Daily streak tracking
* Longest streak tracking
* Daily reset system
* Freeze day support
* Persistent local storage

### XP Reward System

* Earn XP for completing productive actions
* Persistent XP tracking
* Achievement milestones
* Progress-based motivation

### Instant Action Mode

* 5-second countdown
* Immediate action trigger
* XP rewards for starting

### One Minute Rule

* 60-second commitment timer
* Momentum-building technique
* XP rewards on completion

### Reverse Pomodoro

* Focus session timer
* Break session timer
* Session tracking
* XP rewards

### Rewards System

Achievement milestones:

* Beginner (50 XP)
* Consistent (100 XP)
* Focused (250 XP)
* Disciplined (500 XP)
* Unstoppable (1000 XP)
